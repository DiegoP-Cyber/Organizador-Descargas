#  Organizador de Descargas Automático

Script en **Python** para mantener la carpeta de descargas de Windows limpia y categorizada automáticamente.

##  Funcionalidades
- Identifica archivos por su extensión.
- Clasifica automáticamente en carpetas: `Instaladores`, `Comprimidos`, `Imagenes` y `Documentos`.
- Crea las carpetas de destino si no existen.
- Ignora archivos de sistema.

##  Tecnologías
- **Python 3.14**
- Librerías: `os`, `shutil`.
